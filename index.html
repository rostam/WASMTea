WASMTea

A browser-based graph editor powered by Rust and WebAssembly.
Build graphs, run algorithms — no server required.

Graph Editor Large Graph (3D)
Editor

Add vertices and edges with your mouse. Move, delete, and run generators.

Analysis

Connectivity, bipartite, Eulerian paths, diameter, chromatic number, MIS, clique.

Large Graph (3D)

Force-directed 3D layout via ngraph. Supports random, Barabási–Albert, and more up to 500 vertices.
